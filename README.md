# Documenta-o-Semanal-
Documentação semanal

Resumo da Semana

Aula 1: Estrutura e função do Html, Css e JavaScript, como "ligar" o Css ao Html e alguns de seus códigos básicos.

Aula 2: Sintaxe do Html e JavaScript, variáveis, operadores (+,-,/,*...), armazenar dados nas variáveis.

Aula 3: Fizemos uma calculadora de média aritmética com base nas aulas anteriores.

Explicação Técnica:
Aprendemos a usar Let e Const, alguns operadores (+,/)  como funciona a matemática do computador, por exemplo para a calculadora fr média, para que o computador entenda que é necessário somar primeiro e só depois dividir, temos que deixar a soma entre ().

Reflexão Final:
Entender como usar cada variável e usar minha própria técnica, mas prestando atenção eu consegui concluir.
